# Ride Sharing System (Airtribe)

This project is a for Ride Sharing System demonstrating:
- Core OOP principles (inheritance, polymorphism, encapsulation)
- Service-oriented structure
- Design patterns also using like strategy.

## What is included
- `src/main/java/com/airtribe/ridesharingsystem/` - application code
- `src/test/java/` - unit tests case
- `pom.xml` - Maven build
- `docs/` - documentation and setup
